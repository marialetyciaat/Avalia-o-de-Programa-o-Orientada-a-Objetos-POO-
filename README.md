# 🏦 Sistema de Caixa Eletrônico (ATM) - Projeto POO

Este projeto é um simulador de sistema bancário desenvolvido em Python, aplicando os principais pilares da **Programação Orientada a Objetos (POO)**. O sistema permite gerenciar clientes, criar contas (Corrente e Poupança), realizar depósitos, saques e visualizar o histórico de transações.

## 🚀 Como Executar o Sistema

Para rodar o projeto localmente:
1. Certifique-se de estar na pasta raiz do projeto.
2. Execute o seguinte comando no terminal:
   ```bash
   python banco/main.py
